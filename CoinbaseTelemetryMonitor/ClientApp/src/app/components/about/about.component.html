Description

Purpose

This project serves as a simple proof-of-ability demonstration by utilizing Microsoft technologies and services to create a real-time, data driven Azure hosted application.

Process

The ASP.NET Core server application connects to the Coinbase API using a WebSocket. Coinbase transmits real-time price updates every time a match occurs (i.e., when a buyer and seller transact for a given price). The server application then transmits this match price data to the Angular browser application using SignalR where it is visualized using Chart.js.

Potential

Using this technique, historical price data could be saved off and used to compute market indicators (i.e., Bollinger Bands, moving averages, etc.).
